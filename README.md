# Linha do Tempo da IA

Um site criado para mostrar algumas das aplicações da inteligência artifical ao longo dos anos, com o foco em aplicações que habitaram a internet.

# Dependências

Esse site foi feito com **React.js**, e estilizado com CSS puro. 

# Como Rodar

    * Clone o projeto
    * Instale as dependências 
    * Escreva `npm run dev` no terminal
